Detailed Description

Classe base che rappresenta una telecamera nella scena 3D.

  • Estende la classe Node per ereditare le trasformazioni spaziali (posizione, rotazione). Gestisce la matrice di proiezione e calcola la matrice di vista (View Matrix).

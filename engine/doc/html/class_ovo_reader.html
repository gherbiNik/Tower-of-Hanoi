◆ m_materials

std::map<std::string, Material*> OvoReader::m_materials
protected

A map that stores materials parsed from the OVO file.

The key is the material's name (as a string), and the value is a pointer to the corresponding Material object. This map is used to manage and reference materials during the parsing process.

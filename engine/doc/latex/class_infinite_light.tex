\doxysubsection{Detailed Description}
Rappresenta una sorgente luminosa direzionale posta a distanza infinita. 


\begin{DoxyItemize}
\item Questa classe simula una luce i cui raggi colpiscono gli oggetti parallelamente, ideale per simulare sorgenti come il sole. 
\end{DoxyItemize}

\doxysubsection{Detailed Description}
Rappresenta una sorgente luminosa che emette luce uniformemente in tutte le direzioni. 


\begin{DoxyItemize}
\item Simula una sorgente puntiforme (come una lampadina) che illumina la scena sfericamente.
